# 🛡️ Phone Defender - Security Policy

## Data Privacy

### What We Collect
- **Local Device Data Only**
  - App list (for threat detection)
  - Network connections (for monitoring)
  - File metadata (for encryption)
  - User preferences (for settings)

### What We DON'T Collect
- ❌ Personal information
- ❌ Location data
- ❌ Browsing history
- ❌ Call logs or messages
- ❌ App usage analytics
- ❌ Device identifiers

### Data Storage
- **Location:** Device storage only
- **Encryption:** AES-256 for sensitive data
- **Backup:** Optional, user-controlled
- **Retention:** Deleted when app is uninstalled

---

## Security Features

### Encryption
- **Algorithm:** AES-256-CBC
- **Key Management:** Secure keystore
- **File Protection:** Full encryption support
- **Transport Security:** TLS 1.2+

### Permissions
- **Minimal:** Only necessary permissions requested
- **Transparent:** All permissions explained
- **Revocable:** User can disable anytime
- **Scoped:** Limited to security functions

### Code Security
- **Obfuscation:** ProGuard enabled
- **Code Review:** Regular security audits
- **Dependency Updates:** Weekly checks
- **Vulnerability Scanning:** Automated checks

## Reporting Security Issues

If you discover a security vulnerability:

1. **Do NOT** open public GitHub issue
2. **Email:** security@phonedefender.io
3. **Include:** Detailed vulnerability description
4. **Wait:** 48 hours before public disclosure
